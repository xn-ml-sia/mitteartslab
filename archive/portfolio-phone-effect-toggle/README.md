# Portfolio phone effect toggle (archived)

Goo-style SVG checkbox toggle for switching explode effects 1 and 3.

Replaced by a bottom-left cam icon button using `/public/assets/dot-icon/cam.svg`.

## Restore

1. Paste `toggle.html` into `.portfolio-phone__effect-toggle` in `portfolio-phone-showcase.js`.
2. Merge `toggle.css` into `public/portfolio-detail-phone.css` (or link separately).
